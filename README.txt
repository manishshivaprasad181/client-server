1. Open the terminal on linux
2. Execute the following commands, we are compiling the client.c and server.c
	
	$> gcc -o client client.c
	$> gcc -o server server.c

3. Execute the following command to run the exe client and server files in 2 different terminals, ./server-> run the exe server and ./client ->to run the exe client:
	
	# on terminal 1
	$> ./server


	# on terminal 2
	$> ./client
